# **App Name**: Aura Academic

## Core Features:

- Personalized Timetable Creator: An AI-powered tool to generate a customized daily and weekly timetable based on user-inputted classes, study habits, and personal appointments. The AI tool will adapt to changes.
- Assignment & Priority Management: Users can input assignments with due dates, which the system automatically sorts into a prioritized list based on deadlines and user-defined importance.
- Intelligent Reminder System: Provides timely notifications for upcoming assignments, scheduled study sessions, and breaks to keep students on track.
- Focus Mode with Social Media Blocker: Initiate dedicated focus sessions that temporarily block distracting social media applications, ensuring undivided attention on studies.
- Adaptive Study Break Planner: During focus sessions, the AI tool intelligently suggests and schedules short, rejuvenating breaks, prompting users with options to relax and recharge.
- Mental Wellness Activities Portal: Provides quick access to stress-reducing options during breaks, including curated playlists, simple mental health games, or an engaging chatbot for emotional support.
- Social Media Usage Monitor: Allows users to set daily time limits for social media apps and provides gentle reminders when limits are approached or exceeded, without hard blocking unless in focus mode.

## Style Guidelines:

- Light color scheme, emphasizing clarity and calm. Primary: a muted slate blue (#696BA9) conveying professionalism and serenity. Background: a soft, barely-there off-white with a hint of blue-violet (#F4F4F7) for an airy feel. Accent: a vibrant yet sophisticated aqua blue (#4DB9ED) to highlight key actions and information, ensuring good contrast.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly legible characteristics, ensuring ease of reading across all content, from schedules to chatbot interactions.
- Use a set of clean, minimalist line icons that clearly communicate their function. Icons should subtly represent concepts like time, tasks, focus, relaxation, and progress, aligning with the app's calming aesthetic.
- Employ a grid-based, responsive layout with generous white space to promote focus and reduce visual clutter. Information should be presented in digestible blocks, prioritizing easy navigation and glanceable summaries for timetables and priority lists.
- Incorporate subtle, non-distracting animations for transitions between sections, state changes (e.g., entering/exiting focus mode), and feedback on user actions. Animations should be fluid and brief to enhance usability without causing disruption.
